pex¶

这个项目是 .pex 文件和能够创建它们的 pex 工具的家园。pex 还提供了一个类似于 virtualenv 的通用 Python 环境虚拟化解决方案。pex 是 “Python Executable” 的缩写。

简而言之¶

要快速开始构建 .pex 文件，请直接进入 构建 .pex 文件。初次接触 python 打包？请查看 packaging.python.org。

简介和历史¶

pex 包含了 Python 打包和发布库，最初是通过 twitter commons 提供的，但后来被分割成一个单独的项目。pex 最显著的组成部分是 .pex (Python EXecutable) 格式和相关的 pex 工具，它提供了一个通用的 Python 环境虚拟化解决方案，其精神类似于 virtualenv。自 2011 年以来，PEX 文件一直被 Twitter 用于将 Python 应用程序部署到生产中。

要了解更多关于 .pex 格式是什么以及为什么它可能对你有用，请参见 什么是 .pex 文件？ 对于没有耐心的人，还有一个由 Twitter 大学发布的闪电讲座（稍有过时）：WTF is PEX?。要直接建立pex文件，请看 构建 .pex 文件。
